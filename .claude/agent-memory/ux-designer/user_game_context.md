## Project: REQUIEM FORMATION

A vertical scroll shoot 'em up for PC (Steam/Epic) with narrative depth. The player absorbs enemy patterns and sacrifices themselves strategically to build a ghost echo formation.

**Engine**: Unity 6.4 (6000.4.4f1)
**Platform**: PC — Keyboard/Mouse + Gamepad (equal priority, no touch)
**Target Audience**: Hardcore/midcore PC players, age 25-40, plays Hades / Hollow Knight / Ikaruga

## Game Pillars (inform every UX decision)

1. **희생은 선택이다 (Sacrifice is Agency)** — Sacrifice is always deliberate. Accidental sacrifice is a contradiction of the game's thesis, not just bad UX.
2. **잔상이 이야기다 (Echoes Are Narrative)** — Readability of echo formation matters more than visual flourish.
3. **흐름 속의 깊이 (Depth Through Flow)** — Systems should not overwhelm. Progressive disclosure over upfront explanation.
4. **모던 레트로 감성 (Modern Retro)** — 8-bit pixel silhouettes are sacred. Modern effects sit on top, never obscure.

## Anti-Pillars

- NOT bullet hell — evasion is not the primary challenge.
- NOT random roguelite — sacrifice accumulates meaning across runs.
- NOT long grind — dense 1-2 hour emotional arc.

## Key UX Insight for Sacrifice

The Sacrifice input is the game's highest-stakes design decision. An ambiguous or accidentally-triggered sacrifice is not a bad feel issue — it is a philosophical contradiction of the core pillar. Every UX decision about the sacrifice input must be framed in terms of: "does this make sacrifice feel like the player's deliberate choice?"
